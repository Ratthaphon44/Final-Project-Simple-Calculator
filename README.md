# Final-Project-Simple-Calculator

Scientific Calculator is a Single Page Web App that supports basic arithmetic operations and scientific functions including trigonometry (sin, cos, tan), square root, exponentiation, and parentheses handling.

## Features
- Basic Arithmetic: Addition, Subtraction, Multiplication, Division
- Scientific Functions: Sin, Cos, Tan, Square Root (√)
- Advanced Operations: Exponentiation (^), Parentheses ( )
- UI Features: Backspace, Clear, Responsive Grid Layout
- Keyboard Support

## How to run
Open `04_Implementation/index.html` in a browser.
