## User Stories

1. ในฐานะผู้ใช้ ฉันต้องการกดตัวเลขและทศนิยมเพื่อป้อนค่าที่ต้องการคำนวณ
2. ในฐานะผู้ใช้ ฉันต้องการกดปุ่มบวก ลบ คูณ หาร เพื่อสร้าง expression พื้นฐาน
3. ในฐานะผู้ใช้ ฉันต้องการหาค่าทางตรีโกณมิติ (sin, cos, tan) เพื่อใช้ในการคำนวณทางวิทยาศาสตร์**
4. ในฐานะผู้ใช้ ฉันต้องการคำนวณเลขยกกำลัง (x^y) และสแควร์รูท (√) ได้**
5. ในฐานะผู้ใช้ ฉันต้องการใส่วงเล็บ ( ) เพื่อกำหนดลำดับการคำนวณด้วยตัวเอง**
6. ในฐานะผู้ใช้ ฉันต้องการกดปุ่ม = เพื่อดูผลลัพธ์ที่ถูกต้องตามหลักคณิตศาสตร์ (Order of Precedence)
7. ในฐานะผู้ใช้ ฉันต้องการกด C เพื่อเคลียร์ค่าทั้งหมด และกด ⌫ (Backspace) เพื่อลบตัวอักษรล่าสุด
8. ในฐานะผู้ใช้ ฉันต้องการให้ระบบแจ้งเตือน Error หากฉันใส่สมการผิด (เช่น วงเล็บไม่ครบ) หรือหารด้วยศูนย์
9. ในฐานะผู้ใช้ ฉันต้องการใช้งานผ่านคีย์บอร์ดคอมพิวเตอร์ได้เพื่อความรวดเร็ว
10. ในฐานะผู้ใช้ ฉันต้องการ UI ที่แยกสีปุ่มตัวเลขและปุ่มฟังก์ชันออกจากกัน เพื่อให้มองเห็นง่าย
